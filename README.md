# Stability Diagnostics

**One governance law, Theta = I/R, read four ways.**

A family of instruments built from a single idea: a system stays stable, and moves economically, only while its absorption capacity keeps up with the load driving it. Each reads that balance from a different angle, and each is grounded in real, public data with its own live page.

Three instruments predict failure:

- **Instrument 01 - When** (a resource runs out): https://oren-speiser.github.io/cross-domain-earlywarning/
- **Instrument 02 - Where** (the weak coupling seam is): https://oren-speiser.github.io/coupling-seam/
- **Instrument 03 - Whether** (it is about to oscillate): https://oren-speiser.github.io/critical-slowing-down/

A second branch finds waste:

- **Instrument 04 - How well** (it moves, cost of transport): https://oren-speiser.github.io/cost-of-transport/

**Live hub:** https://oren-speiser.github.io/stability-diagnostics/

This page is a gateway: it states the shared law and links to the instruments. Each instrument page states its own scope and honest limits.
